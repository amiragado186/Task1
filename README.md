Standard Numeric Format  
1. why the output of this Equation = $30.00?
Because of using Standard Numeric format at the code (at this part {X + Y:C})
'C' means covert the string to currency (default currency at the system).

2. what is its benefit?
make it easy and quick to format common numeric types like currency, percent, decimal, etc.
instead of custom patterns.
